# 要件定義書

**プロジェクト名**: Claude Code × YouTubeショート動画まとめサイト
**バージョン**: 1.0
**作成日**: 2026-02-15

---

## 1. 概要

あきらパパ様のYouTubeチャンネルに投稿しているClaude Code関連ショート動画を一覧・視聴できるまとめサイト。

## 2. 機能要件

### 2.1 動画一覧表示

- 動画をカード形式でグリッド表示
- サムネイル、タイトル、投稿日、再生時間を表示
- レスポンシブグリッド（PC: 3列、タブレット: 2列、スマホ: 1列）

### 2.2 動画再生

- カードクリックで動画詳細ページ or モーダルで再生
- YouTube埋め込みプレイヤー（iframe）を使用

### 2.3 フィルタリング・検索

- カテゴリタグでフィルタリング（例: "基本操作", "実践", "Tips"）
- テキスト検索（タイトル・説明文）

### 2.4 ソート

- 新着順（デフォルト）
- 人気順（再生回数 ※将来対応）

### 2.5 ヘッダー・フッター

- サイトタイトル・ロゴ
- YouTubeチャンネルへのリンク
- SNSリンク（X: @akira_papa_IT）

## 3. 非機能要件

### 3.1 パフォーマンス

- Lighthouse Performance 90+
- 初期表示 2秒以内

### 3.2 レスポンシブ対応

- モバイル、タブレット、デスクトップ対応
- ブレークポイント: 640px / 1024px

### 3.3 SEO

- メタタグ設定
- OGP画像
- サイトマップ

### 3.4 アクセシビリティ

- キーボードナビゲーション
- alt属性
- WCAG 2.1 AA準拠

## 4. データ構造

```typescript
interface Video {
  id: string;           // YouTube動画ID
  title: string;        // タイトル
  description: string;  // 説明文
  thumbnail: string;    // サムネイルURL
  publishedAt: string;  // 投稿日
  duration: string;     // 再生時間
  tags: string[];       // カテゴリタグ
  viewCount?: number;   // 再生回数（将来対応）
}
```

## 5. ページ構成

| ページ | パス | 説明 |
|--------|------|------|
| トップ | `/` | 動画一覧 + フィルタ |
| 動画詳細 | `/video/[id]` | 動画再生 + 詳細情報 |

## 6. 制約

- YouTube Data API のクォータ制限（初期はJSON管理で回避）
- ショート動画のアスペクト比（9:16）に対応
