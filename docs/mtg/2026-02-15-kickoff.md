## 議題

### 1. プロジェクト概要

あきらパパ様のYouTubeチャンネル「あきらパパのAI活用学習部屋」に投稿している **Claude Code関連のショート動画** を一覧・視聴できるまとめサイトを作成する。

### 2. ゴール

- ショート動画を手軽に一覧できる
- サイト上で動画を視聴できる
- カテゴリ別・タグ別にフィルタリングできる
- レスポンシブ対応（スマホ・PC）
- Vercelでデプロイ

### 3. 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| フレームワーク | Next.js (App Router) | あきらパパ様の技術スタック |
| 言語 | TypeScript | 型安全 |
| スタイリング | Tailwind CSS | 高速開発 |
| デプロイ | Vercel | Next.jsとの親和性 |
| データ管理 | JSON（静的） | 初期はシンプルに |

### 4. 動画データの管理方針

- 初期: `data/videos.json` にハードコード
- 将来: YouTube Data API v3 で自動取得も検討

### 5. スケジュール

1. ドキュメント作成（会議・要件定義・設計）
2. 実装
3. レビュー・修正
4. ビルド・動作確認
5. GitHub push → Vercelデプロイ
